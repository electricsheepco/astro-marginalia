# Credits

Astro Marginalia is built with Astro, TypeScript, Tailwind CSS, Pagefind, and plain CSS.

The default typography uses Noto Serif, Cormorant Garamond, and JetBrains Mono from Google Fonts. Replace them in `src/styles/global.css` for your own language or publication system.

Demo citations use example URLs and must be replaced before publishing a real archive.
